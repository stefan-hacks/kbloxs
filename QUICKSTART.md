# Quick Start Guide

## Installation (30 seconds)

```bash
./install.sh
source ~/.bashrc
```

## Usage

### Long form (always works)
```bash
blx ls -la
blx ps aux
blx git status
```

### Short form (recommended - just prefix with 'b')
```bash
b ls -la
b ps aux
b git status
```

## Examples

```bash
# File listing
b ls -la

# Process list (auto-formatted as table)
b ps aux

# Git status
b git status

# Docker
b docker ps

# System info
b df -h
b systemctl status

# Any command!
b <your-command> [args...]
```

## That's It!

Just prefix any command with `b` to get beautiful block-styled output with automatic table formatting!
